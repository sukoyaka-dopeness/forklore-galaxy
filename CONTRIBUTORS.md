### Session 3 — ChatGPT
**Contributions:**
- Self-loop policy formalized (mini-loop story = valid structure)

- Node persistence policy (no deletion, visibility control only)
- Cross-project connection design (`repo_url::node_id` format)
- Name candidates added: StoryGraph, Narrative Multiverse, Living Narrative, Story Genome, **Forkverse**
- Protocol candidates: NGP (Narrative Graph Protocol), ONF (Open Narrative Format)
- DAO designated as future-phase concept

---

### Session 4 — Claude
**Contributions:**
- `bridge_to` directionality spec (`directed: true/false`)
- Node character limit confirmed (1,200 Japanese characters equivalent)
- Isolated node policy: `status: "floating"` — same spec as external reference
- Multiple genesis policy: contradictions between genesis nodes = multiverse, welcomed
- Instance policy finalized: each instance operator sets their own rules, no central control
- Translation UI initial spec: external link buttons (DeepL, Google Translate, Microsoft Translator)
- GitHub structure confirmed: personal repository hub + fork model
- `relations.json` ownership: each fork owner manages their own
- Folklore seed node types defined: PD verbatim / AI-written from PD / AI adaptation of PD essence
- Folklore selection memo produced (separate document)

---
